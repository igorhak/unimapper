Get started

Minimal requirements

You don't need any framework, extensions or third part libraries. Everything you need is PHP 5.4+ and some nice application for the use of this tool.

Easy to use

There

Fully extendable

We do not tell you strictly how to use it. There are some built-in queries, mappers and framework extensions, but if you need something more, just write your own extension based on official API.

ORM or models?

It's your decision, but you can try both of them. There are some entities, repositories or you can use only raw data mapping.

Collect all your data

Data collection has never been easier before. You don't need to know where are your data stored or think about how to mixed them together. Actually you only write something like descriptions (entities) and work with it.
